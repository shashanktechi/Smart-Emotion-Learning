# Smart Emotion-Based Learning Platform

## Overview

Smart Emotion-Based Learning Platform is an AI-powered web application that detects a student's emotions through facial expressions and provides personalized learning content.

## Tech Stack

### Frontend

- React (Vite)

### Backend

- Node.js
- Express.js

### AI

- Python
- Flask
- OpenCV
- TensorFlow

### Database

- MySQL

### Deployment

- Vercel
- Render
- AWS RDS

## Project Structure

```plaintext
Smart-Emotion-Learning
│
├── frontend
├── backend
├── ml-api
├── database
└── docs
```

## Features

- User Authentication
- Emotion Detection
- Personalized Learning
- Student Dashboard
- Progress Tracking
- Admin Panel
